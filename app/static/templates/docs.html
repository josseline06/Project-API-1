Documentación API - Duckbook

Registrar un usuario nuevo

/api/users

  • POST

(Enviados mediante x-www-form-urlencoded)

  • Obligatorios
    • Apodo y método de ingreso del usuario
      • string
    • Contraseña del usuario para ingresar al sistema
      • string
    • Nombre del usuario nuevo
      • string
    • Apellido del usuario nuevo
      • string
  • Mensaje
    • Usuario agregado
    • message:'User created!'
    • Usuario no agregado
  • JSON

    • /api/users

Listar un Usuario

/api/users/:username

(El usuario debe estar autenticado)

  • GET
  • Obligatorios
    • :username
      • string
  • Mensaje
    • User
    • ERROR
  • JSON

    • /api/user/juan

Autenticar un usuario

/api/authenticate

  • POST

(Enviados mediante x-www-form-urlencoded)

  • Obligatorios
    • Apodo y método de ingreso del usuario
      • string
    • Contraseña del usuario para ingresar al sistema
      • string
  • Mensaje
    • success: booleano que retorna true si se encontró el usuario, false en caso contrario
    • message:'Enjoy your token!'
    • token: string_que_sera_token
  • JSON

    • /api/authenticate

Listado de Palabras

/api/words

(El usuario debe estar autenticado)

  • GET
  • Obligatorios
    • Ninguno
  • Mensaje
    • Listado de todas las palabras
  • JSON

    • /api/words

Listar una palabra

/api/words/:word_text

  • GET
  • Obligatorios
    • :word_text
      • string
  • Mensaje
    • Palabra
    • null
  • JSON

    • /api/words/palabra

Agregar una palabra

/api/words

  • POST
  • Obligatorios
    • text
      • string
    • feeling
      • número entero
    • word_type
      • string
  • Mensaje
    • Palabra agregado
    • message:'Word created!'
    • Palabra no agregado
  • JSON

    • /api/words

Actualizar una palabra

/api/words/:word_text

  • PUT
  • Obligatorios
    • :word_text
      • string
    • feeling
      • número entero
  • Mensaje
    • Palabra actualizada
    • message:'Word updated!'
    • Palabra no actualizada
  • JSON

    • /api/words

Eliminar una palabra

/api/words/:word_text

  • DELETE
  • Obligatorios
    • :word_text
      • string
  • Mensaje
    • palabra eliminada
    • message:'Word deleted!'
    • palabra no eliminada
  • JSON

    • /api/words/palabra

Listar un Tweet

/api/users/:username/tweets/:tweet_id

  • GET
  • Obligatorios
    • :username
      • string
    • :tweet_id
      • string
  • Mensaje
    • Tweet
    • ERROR
  • JSON

    • /api/users/juan/tweets/51d27328a

Agregar un Tweet

/api/users/:username/tweets

  • POST
  • Obligatorios
    • :username
      • string
    • content
      • string
  • Mensaje
    • Tweet agregado
    • success: booleano que es true si se puedo agregar, false en caso contrario
    • message:'Tweet created!'
    • message: 'Inexistent word(s) in tweet'
    • Tweet no agregado
  • JSON

    • /api/users/juan/tweets

Actualizar un Tweet

/api/users/:username/tweets/:tweet_id

  • PUT
  • Obligatorios
    • :username
      • string
    • :tweet_id
      • string
    • content
      • string
  • Mensaje
    • Tweet actualizado
    • success: booleano que es true si se puedo actualizar, false en caso contrario
    • message:'Tweet created!'
    • message: 'Inexistent word(s) in tweet'
    • Tweet no actualizado
  • JSON

    • /api/users/juan/tweets/51d27328a

Eliminar un Pastie

/api/users/:username/tweets/:tweet_id

  • DELETE
  • Obligatorios
    • :username
      • string
    • :tweet_id
      • string
  • Mensaje
    • tweet eliminado
    • tweet no eliminado
  • JSON

    • /api/users/juan/tweets/51d27328a

--NOTAS:

  • Los IDs son únicos y hexadecimales

  • Las palabras son mostradas de la mas antiguo a la mas reciente (creado o actualizado)

  • Si tiene conocimiento de alguna otra herramienta para verificar el funcionamiento del servicio la puede usar, por ejemplo: el comando curl de UNIX o Postman
